Find the Object (music looping doesn't work in chrome)

  • High Concept: Find an object in the image that starts with the given letter.
  • Player Experience Goals: Investigation and learning.
  • Genre: Point and click.
  • Game play: The player is instucted to click on an object that starts with the given letter and will pass the level if they do. If they click on the wrong object, the game will tell them the name of that object.
  • Prototype goal: 4 year old appropriate
  • Features: Clicking on different objects in an image to find and learn the one that starts with a certain letter and starting letter is random each time.
  • Setting/Premise: There is none. It's a kids game.
  • Story: None.
  • Target Audience: 4 year olds.
  • Play time: A few minutes.
  • Strategies: Knowing words.
